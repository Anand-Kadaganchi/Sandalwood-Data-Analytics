# Sandalwood Data Analytics Project

This project focuses on **Python-based Data Analytics** for agricultural optimization. It processes and analyzes large-scale sandalwood yield data to provide actionable business insights.

## Project Overview
* **Goal**: Perform Exploratory Data Analysis (EDA) and build an interactive analytical tool to assess sandalwood growth factors.
* **Dataset**: Utilized a real-world dataset of 200,000+ records, performing extensive data cleaning and feature engineering.
* **Key Features**: Analyzed 11+ variables including soil pH, temperature, and fertilizer frequency to determine high-yield potential.

## Tech Stack
* **Language**: Python (Pandas, NumPy)
* **Visualization**: Streamlit
* **Deployment**: Interactive Dashboards for decision support

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install streamlit pandas joblib`
3. Run the application: `streamlit run app.py`
